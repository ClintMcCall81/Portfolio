Music App

This was a JavaScript based music player I did as a test of my skills since starting to learn Javascript, This taught me a lot about arrays and multiple other functions that could be used in real life applications. This didnt use Bootstrap or Jquery since this was my first real project using JavaScript. The music player plays 3 songs currently and allows you to skip a song, play and pause a song, as well as adjust the volume.

Process

The process applied to this project was normally how I would approach a web design page, Structure and Functionality comes first, styling comes second. This approach has saved me from tons of mistakes and always gives me great looking results.
